# Affordable Mechanic - Business Website

A modern, responsive React web application for an affordable mechanic service business in Mississauga, near Pearson Airport.

## Features

- **Service Listing**: Showcase all automotive services offered
- **Car Brands**: Display all 33 supported car brands
- **Contact Form**: Professional contact form with validation for quotes and appointments
- **Responsive Design**: Fully responsive design that works on all devices
- **Modern UI**: Clean, professional design with smooth animations

## Services Offered

- Regular maintenance (oil changes & tune-ups)
- Brake & suspension service
- Exhaust & gasket repairs
- Diagnostics & inspections (leaks, electrical)
- Engine/transmission swaps

## Technology Stack

- React 18+
- Vite (build tool)
- Modern CSS (CSS Grid, Flexbox)
- Responsive design patterns

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/

│   ├── Services.jsx     # Services listing

│   ├── ContactForm.jsx  # Contact form with validation
│   └── Footer.jsx       # Footer with location info
```
